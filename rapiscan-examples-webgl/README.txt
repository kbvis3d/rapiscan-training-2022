Using the WebGL Course Example Code
-----------------------------------

- Right-click on the Google Chrome shortcut and edit the path in "Target". Use this shortcut to launch Chrome with file access enabled (for loading textures and models locally)
- Open index.html which lists the WebGL examples

System Requirements:
--------------------
- Recent version of Google Chrome. These example were tested on version 32.0.1700.107
- Windows 7/8 computer with graphics support for OpenGL 2.0 or greater
